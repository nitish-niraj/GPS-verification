    Available Endpoints

    POST /api/v1/validate-image-location

    Upload an image to extract GPS coordinates and validate location against LPU campus boundaries

    Request: multipart/form-data with image file
    Response: GPS coordinates, validation status, zone information
    POST /api/v1/validate-coordinates

    Validate latitude/longitude coordinates directly without an image

    Request: JSON with latitude and longitude
    Response: Validation status and zone information
    GET /api/v1/zones

    List all configured validation zones (LPU campus areas)

    Response: List of zones with boundaries and metadata
    GET /api/v1/health

    Check API health status and service availability

    Response: Health status, OCR availability, zones loaded

    Key Features

    • ✅ EXIF GPS Extraction: Extract coordinates from camera metadata
